## Getting Started

1. **Clone this repository**

    ```bash
    git clone https://github.com/TommyWellsRocks/astonware.git astonware
    ```

    ```bash
    cd astonware
    ```

2. **Install dependencies**

    ```bash
    pnpm install
    ```

3. **Run development server**

    ```bash
    pnpm dev
    ```

4. **[Open Astonware](http://localhost:3000)**
